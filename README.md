# marbis-practical

## Project setup

```
npm install
```

### Code Formatting
To ensure consistent code formatting throughout this project, simply execute the following command:
```
npm run format
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your unit tests

```
npm run test:unit
```

### Lints and fixes files

```
npm run lint
```
